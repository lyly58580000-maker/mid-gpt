# v1.1.0 AI 工作台升级说明

## 版本判定

**v1.1.0** — 在 v1.0.1 基础上新增用户画像、项目记忆、回答模式、场景模板与动态 Prompt 组装，属于功能性 MINOR 版本，不破坏原有聊天/余额/生图流程。

---

## 数据库迁移（必须先执行）

```bash
cd sheyan-ai
npx prisma db push
npx prisma db seed
```

新增表：`UserProfile`、`Project`、`ProjectMemory`、`ConversationSummary`、`SceneTemplate`、`AnswerMode`  
`Conversation` 新增字段：`projectId`、`answerModeSlug`、`sceneTemplateSlug`、`useMemory`

- 10 个场景模板 seed

## 如何测试

1. `npx prisma db push && npx prisma db seed`
2. `npm run dev`，登录演示账号
3. 顶部工具栏选择「深度分析」模式，提问复杂问题 → 回答应更详细
4. 新建项目并添加记忆，选中项目后提问 → 回答应引用项目信息
5. 打开「画像」填写偏好 → 后续回答应体现偏好
6. 问「这个专利能不能申请」类问题 → 应出现边界提示
7. 说「以后都给我简洁点」→ 用户画像 `outputPreference` 应更新
8. 原有聊天、切换对话、并行生成、余额扣点应仍正常

## 已知限制 / 风险

1. 记忆提取为**规则型 MVP**，非 LLM 二次分析，精度有限
2. 需执行 `prisma db push`，否则新 API 会报错
3. 本地 dev 运行时若 `prisma generate` 报 EPERM，需先停 dev 再 generate
4. **尚未推送到 GitHub**，等你验收后再发布 v1.1.0
